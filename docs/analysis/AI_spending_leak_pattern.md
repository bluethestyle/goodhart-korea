## C. 재출연 단계 추적 (데이터 한계)

```

재출연(출연기관 → 재단/개별과제 → 연구자) 단계는 다음 데이터에서만 식별 가능:
  · 국고보조금통합관리시스템 (KODAS, 신청 후 접근)
```
