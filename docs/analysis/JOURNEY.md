## 0. 한 페이지 요약

**연구 질문**: 정부 재정 집행이 KPI에 적응한 결과, 분야별로 어떻게 다른 시점 패턴을 보이는가?

**이론적 토대**: 굿하트 법칙(Goodhart 1975, Strathern 1997) — 측정 지표가 목표가 되는 순간 좋은 지표이기를 멈춘다. NPM 비판 학파(Bevan & Hood 2006)는 4가지 게임화 패턴을 식별했고, 이 중 **Endpoint clustering**(평가 시점 직전 실적 몰아주기)은 정부 예산 집행에서 가장 관찰되기 쉽다.

**핵심 발견**: 한국 정부 16대 분야 사이의 **연주기 게임화 강도가 6배 차이**(0.124 ~ 0.746)를 보이며, 이는 사업 단위 분할도·KPI 평가 강도·행정 재량의 함수로 설명된다.

**메인 가설**:
- **H1 (행정 메커니즘)**: 게임화 강도 ∝ 사업 단위 분할도 × KPI 평가 × 행정 재량
- **H2 (정책 효과 Decoupling)**: 게임화 강도가 높은 분야일수록 정책 결과지표 개선이 더디다

**다음 단계**: KOSIS·KIPRIS·OECD 등에서 분야별 결과변수 다운로드 → 패널 FE 회귀로 H2 검증.

## 1. 연구 배경

한국 정부 예산은 매년 국회 심의를 거쳐 편성되고, 각 부처는 회계연도 내 집행을 의무로 한다. 미집행 예산은 다음 해 편성에서 불리하게 작용하므로, 부처는 집행률을 KPI로 인식하게 된다.

이러한 KPI는 본래 재정 효율성을 측정하기 위한 도구였으나, 시간이 지나며 **측정 자체가 목적이 되는 굿하트 효과**가 발생한다. 감사원은 매년 결산검사보고서에서 "12월 일제 집행", "이월·불용 회피 목적의 무리한 집행" 등을 지적해왔다.

본 분석은 다음 질문을 다룬다:
1. 한국 정부 재정 집행에서 굿하트 적응의 흔적이 시계열 패턴으로 관찰 가능한가?
2. 분야별로 게임화 강도가 다른가? 그 차이의 구조적 원인은 무엇인가?
3. 게임화 적응이 강한 분야는 실제 정책 결과지표에서 어떤 패턴을 보이는가?

## 4. 이론적 토대 — 굿하트 법칙

### 4.1 학술적 정식화

**Goodhart (1975)**: "Any observed statistical regularity will tend to collapse once pressure is placed upon it for control purposes."

**Strathern (1997) 일반화**: "When a measure becomes a target, it ceases to be a good measure."

**Manheim & Garrabrant (2018) 4가지 변형**:
1. Regressional Goodhart
2. Causal Goodhart
3. Extremal Goodhart
4. **Adversarial Goodhart** ← 우리 분석의 핵심

### 4.2 Bevan & Hood (2006) — NPM 4가지 게임화 패턴

영국 NHS 사례에서 정식화된 4가지 패턴:

| 패턴 | 정의 | 우리 데이터에서의 매칭 |
|---|---|---|
| Threshold gaming | 임계값 근처 분포 비정상 | 100% 집행률 직전 bunching |
| Output substitution | 측정 안 되는 것 희생 | 본예산 → 추경 이전 |
| Ratchet effect | 작년 KPI → 다음 해 결정 | 12월 미집행의 학습 회피 |
| **Endpoint clustering** | **평가 시점 직전 실적 몰아주기** | **12월/Q1 집중 집행 ★** |

### 4.3 보조 사례 — 코브라 효과(Cobra Effect)

영국령 인도에서 코브라 1마리당 보상금을 지급하자, 사람들이 코브라를 사육하기 시작했다. 보상이 폐지되자 사육 코브라를 풀어줘 야생 개체수가 오히려 증가했다. 인센티브 설계의 역효과로, 게임화의 본질을 직관적으로 보여주는 사례.

### 5.1 단순 비율 분석 (분류 정제 이전)

`monthly_exec` 데이터에 단순 정의 적용:

```
12월 집중도 = 12월 EP_AMT / 연간 총 집행
정상(균등) 기준선 = 8.33%

연도   정부 전체 12월 집중도
2020   10.09%
2024   10.02%
2025   10.11%
```

정부 전체 평균은 균등 대비 +1.7%p로 약함. 그러나 분기 단위로 보면 다른 그림:

```
연도   Q1     Q2     Q3     Q4
2024   27.5   25.1   24.0   23.4
2025   27.0   24.8   26.8   21.4
```

Q1이 균등 25% 대비 +2~5%p 초과. **12월 몰아쓰기보다 Q1 빨리쓰기가 더 두드러짐**.

**관찰**: 분야별 12월 집중도 격차가 8배(33% vs 4%). 일부 분야는 명확한 시점 집중을 보이고 일부는 균등 패턴. 단순 비율 분석으로 1차 패턴은 보임.

## 6. 시행착오 1 — 회계거래 분류의 어려움

### 6.1 문제 인식

분야별 12월 집중도 분석에서 **상위 100% 집중 사업들이 모두 회계 거래(전출·예탁·정산)** 임을 발견. 진짜 정책 사업의 게임화를 측정하려면 회계 거래를 분리해야 함.

### 6.2 시도 1 — 키워드 필터 (넓음)

`ACTV_NM`에 "전출/예탁/이전지출/상환/보전/정산" 등 키워드 매칭:
- 회계거래 226개 / 1,492조 (74.5%)
- 정책사업 2,529개 / 512조 (25.5%)

**문제점**: 지방교부세·기초연금 등 정책 본질의 사업이 회계 키워드로 같이 빠짐.

### 6.3 시도 2 — 좁힌 키워드

순수 자금이동만 분리 ("전출금/타계정/예탁/여유자금운용"):
- 회계거래 56개 / 1,053조 (52.5%)
- 정책사업 2,697개 / 951조 (47.5%)

지방교부세 복귀.

### 6.4 시도 3 — 편성목(CITM) 객관 분류

`expenditure_item`의 `CITM_NM` 기반 객관 분류:
```
CITM = '전출금등'              → 회계거래
CITM = '자치단체이전'          → 지방이전(정책)
나머지 (인건비/운영비/출연금/...) → 정책사업
```

결과:
- 정책사업 1,891개 / 243조
- 지방이전(정책) 188개 / 215조
- 회계거래 97개 / 92조

**키워드 vs CITM 일치도 검증**:
- 키워드는 정책으로 분류했으나 CITM은 회계거래로 본 사례 89건 (회계간거래·예비비 등)
- → CITM이 89건의 회계거래를 추가로 잡아냄, 더 정확

### 6.5 핵심 정정 — Q1 빨리쓰기 가설의 약화

CITM 기반 정책사업만으로 분기 집중도 재계산:

```
연도   Q1     Q2     Q3     Q4
2020   25.9   27.9   21.6   24.5
2024   26.3   24.7   24.8   24.2   ← 거의 균등!
2025   26.2   24.4   27.4   22.1
```

**충격적 정정**: 이전 분석의 "Q1 빨리쓰기 30~35%"는 **자치단체이전(지방교부세) 분기 분배 패턴**의 영향이었음. 정책사업만 보면 Q1 26%로 균등에 가까움.

### 6.6 시행착오의 학술적 의의

분류 노이즈에 따라 결과가 크게 흔들림은 단순 비율 분석의 한계를 명확히 보여줌. **더 강건한 분석 방법론으로 주기성 분해(FFT + STL + Coherence)** 도입 필요.

## 7. 가설 확장 — 주기성 분석

### 7.1 동기

단순 12월 집중도는 한 점만 측정. 시계열 전체의 주파수 성분을 분해하면:
- 정상 정책 흐름 = 트렌드 위주, 약한 계절성
- 게임화된 사업 = 강한 12월/Q1 등 특정 주기 amplitude

이는 분류 노이즈에 강건하고 학술적으로 단단한 측정 방식.

### 7.2 분석 도구

| 도구 | 출처 | 적용 |
|---|---|---|
| FFT | Cooley-Tukey (1965) | 주파수 amplitude 측정 |
| STL Decomposition | Cleveland et al. (1990) | Trend + Seasonal + Residual 분해 |
| Coherence | Welch's method | 두 시계열의 주파수 영역 상관성 |
| Hierarchical Clustering | scipy | Coherence 매트릭스 → 자동 그룹화 |
| Mann-Whitney U Test | 비모수 검정 | 분포 차이 통계 검정 |

### 7.3 3-Tier 분석 구조

| Tier | 단위 | 도구 | 목적 |
|---|---|---|---|
| **Tier 1** | 분야 16개 | STL + FFT | 거시 비교, 시각화 |
| **Tier 2** | 부문 76개 | Coherence + Clustering | 주기적 연관성 발견 |
| **Tier 3** | 단위사업 ~1,600개 (100억+) | 게임화 지수 분포 | 정량 검정 |

## 8. 핵심 발견 — 분야별 게임화 격차 6배

### 8.1 Tier 1: 분야별 연주기 강도

`amp_12m_norm` = 연주기(12개월) FFT amplitude / 평균 — 정규화된 진폭

```
High-Goodhart (강한 주기)         Low-Goodhart (정상)
─────────────────                ─────────────────
과학기술        0.746              국방          0.172
국토·지역      0.607              사회복지      0.166
환경           0.601              공공질서      0.137
보건           0.537              일반·지방행정  0.124
문화·관광      0.467
교통·물류      0.449
산업·중기      0.416
```

**관찰**: 분야별 격차 6배 (0.124 vs 0.746). 인프라·환경·R&D 분야가 강한 주기, 의무 지출 분야(국방·복지·일반행정)가 정상.

### 8.2 구조적 해석

| | High-Goodhart 분야 | Low-Goodhart 분야 |
|---|---|---|
| 사업 형태 | 사업 단위 분할 (건설·R&D·보조금) | 자동 정산·이전 (인건비·교부금·연금) |
| KPI 적용 | 사업별 평가 강함 | 의무 지출 (평가 약함) |
| 행정 재량 | 큼 (시점·금액 조정) | 작음 |
| 대표 사업 | 건설 발주, R&D 출연, 보조금 | 공무원 인건비, 기초연금, 지방교부세 |

→ **굿하트에 취약한 분야 = 사업 단위 분할 + KPI 평가 + 행정 재량 큼** (3박자 가설)

### 8.3 Tier 2: "주기적 연관성"의 자동 발견

76개 부문에 대해 12월 주기 coherence 행렬을 산출하고 hierarchical clustering 적용 → 8개 클러스터로 분해.

가장 흥미로운 클러스터:
```
C6 클러스터 (3개 부문) — 같은 주기 패턴:
  - 공적연금
  - 보건의료
  - 중소기업및소상공인육성
```

정책 카테고리상 무관해 보이지만 **행정 처리 주기가 동일**. 정기 정산·지급 시점이 일치하는 사업군. 이는 분야 코드와 무관하게 행정 패턴을 발견하는 데이터 주도 분류.

### 8.4 Tier 3: 단위사업 게임화 지수 분포

100억 이상 단위사업 1,608개에 대해 사업별 게임화 지수(`amp_12m_norm`) 산출.

```

─────────────────────────
교통및물류       0.926
산업·중기        0.868
환경            0.864
농림수산         0.782
국토·지역       0.778

```

Tier 1 분야 합계 시계열과 약간 다른 순위 — 사업 단위에서는 인프라 분야가 더 강함. 이는 분야 합계가 일부 큰 사업의 영향을 받기 때문이며, 사업 단위 중앙값이 더 강건한 측정.

### 8.5 통계 검정

**Mann-Whitney U test** — 분야 그룹 간 게임화 지수 분포 차이:
- 전반적으로 분야별 분포 차이는 명확 (boxplot 확인)
- 일부 그룹 비교에서는 통계적 유의성 약함 → 단일 분야로 좁히는 가설보다 **분야간 격차의 구조적 설명**이 더 강함

## 9. 메인 가설 정식화

```
[H1] 행정 메커니즘
  Game_i = f(사업단위분할도_i, KPI평가강도_i, 행정재량_i)
  
[H2] 정책 효과 (Decoupling)
  ∂Outcome_i / ∂Game_i < 0
  (게임화 강도가 높을수록 결과지표 개선이 더디다)
```

### 9.1 H1의 입증 — 우리 데이터로 완결

- 분야별 amp_12m_norm 6배 격차 ✓
- 사업 형태와의 상관관계 (인프라·R&D > 의무지출) ✓
- 사업 규모와의 비선형 관계 (작은 사업 + 거대 사업 모두 게임화) ✓

### 9.2 H2의 입증 — 외부 결과변수 필요

다음 단계 분석:
```
패널 FE 회귀:
  Outcome_it = α + β·Game_i + γ_t + δ_i + ε
  
β < 0 검증 = "게임화가 결과를 갉아먹는다"
```

## 11. 한계 및 주의

### 11.1 데이터 한계
- **편성 vs 결산 차이**: 본예산·추경·월별 집행 보유, 결산(이월·불용)은 일부만
- **5년 시계열**: 60개월이지만 1년 주기는 5회만 관측 → 통계적 검정력 한계
- **분야 분류 의존성**: 16대 분야는 행정 분류이지 행동 분류가 아님

### 11.2 인과추론 한계
- 한국 단일 정부 정책 → DID 통제군 부재
- 처치(KPI 도입) 시점 명확하지 않음 (2007년 디지털예산회계 도입 후 점진적)
- 역인과·공변량 누락(코로나 등) 통제 어려움

### 11.3 측정 한계
- "KPI 평가 강도"·"행정 재량"의 직접 측정 불가 → 분야 더미 사용
- amp_12m_norm은 12개월 주기 진폭만 측정 (다른 주기는 별도)

### 11.4 정책 함의의 신중함
- 게임화가 강하다 ≠ 행정이 비효율적이다
- 일부 분야의 시점 집중은 사업 특성(공사 완료 시점·계절성·법정 의무)에서 비롯
- 정책 제언은 게임화 식별 + 결과변수 Decoupling 입증 후에만
