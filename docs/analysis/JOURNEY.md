## 0. 한 페이지 요약

**연구 질문**: 정부 재정 집행이 KPI에 적응한 결과, 분야별로 어떻게 다른 시점 패턴을 보이는가?

**이론적 토대**: 굿하트 법칙(Goodhart 1975, Strathern 1997) — 측정 지표가 목표가 되는 순간 좋은 지표이기를 멈춘다. NPM 비판 학파(Bevan & Hood 2006)는 4가지 게임화 패턴을 식별했고, 이 중 **Endpoint clustering**(평가 시점 직전 실적 몰아주기)은 정부 예산 집행에서 가장 관찰되기 쉽다.

**핵심 발견**: 한국 정부 16대 분야 사이의 **연주기 게임화 강도가 6배 차이**(0.124 ~ 0.746)를 보이며, 이는 사업 단위 분할도·KPI 평가 강도·행정 재량의 함수로 설명된다.

**메인 가설**:
- **H1 (행정 메커니즘)**: 게임화 강도 ∝ 사업 단위 분할도 × KPI 평가 × 행정 재량
- **H2 (정책 효과 Decoupling)**: 게임화 강도가 높은 분야일수록 정책 결과지표 개선이 더디다

**다음 단계**: KOSIS·KIPRIS·OECD 등에서 분야별 결과변수 다운로드 → 패널 FE 회귀로 H2 검증.

## 1. 연구 배경

한국 정부 예산은 매년 국회 심의를 거쳐 편성되고, 각 부처는 회계연도 내 집행을 의무로 한다. 미집행 예산은 다음 해 편성에서 불리하게 작용하므로, 부처는 집행률을 KPI로 인식하게 된다.

이러한 KPI는 본래 재정 효율성을 측정하기 위한 도구였으나, 시간이 지나며 **측정 자체가 목적이 되는 굿하트 효과**가 발생한다. 감사원은 매년 결산검사보고서에서 "12월 일제 집행", "이월·불용 회피 목적의 무리한 집행" 등을 지적해왔다.

본 분석은 다음 질문을 다룬다:
1. 한국 정부 재정 집행에서 굿하트 적응의 흔적이 시계열 패턴으로 관찰 가능한가?
2. 분야별로 게임화 강도가 다른가? 그 차이의 구조적 원인은 무엇인가?
3. 게임화 적응이 강한 분야는 실제 정책 결과지표에서 어떤 패턴을 보이는가?

## 4. 이론적 토대 — 굿하트 법칙

### 4.1 학술적 정식화

**Goodhart (1975)**: "Any observed statistical regularity will tend to collapse once pressure is placed upon it for control purposes."

**Strathern (1997) 일반화**: "When a measure becomes a target, it ceases to be a good measure."

**Manheim & Garrabrant (2018) 4가지 변형**:
1. Regressional Goodhart
2. Causal Goodhart
3. Extremal Goodhart
4. **Adversarial Goodhart** ← 우리 분석의 핵심

### 4.2 Bevan & Hood (2006) — NPM 4가지 게임화 패턴

영국 NHS 사례에서 정식화된 4가지 패턴:

| 패턴 | 정의 | 우리 데이터에서의 매칭 |
|---|---|---|
| Threshold gaming | 임계값 근처 분포 비정상 | 100% 집행률 직전 bunching |
| Output substitution | 측정 안 되는 것 희생 | 본예산 → 추경 이전 |
| Ratchet effect | 작년 KPI → 다음 해 결정 | 12월 미집행의 학습 회피 |
| **Endpoint clustering** | **평가 시점 직전 실적 몰아주기** | **12월/Q1 집중 집행 ★** |

### 4.3 보조 사례 — 코브라 효과(Cobra Effect)

영국령 인도에서 코브라 1마리당 보상금을 지급하자, 사람들이 코브라를 사육하기 시작했다. 보상이 폐지되자 사육 코브라를 풀어줘 야생 개체수가 오히려 증가했다. 인센티브 설계의 역효과로, 게임화의 본질을 직관적으로 보여주는 사례.

### 5.1 단순 비율 분석 (분류 정제 이전)

`monthly_exec` 데이터에 단순 정의 적용:

```
12월 집중도 = 12월 EP_AMT / 연간 총 집행
정상(균등) 기준선 = 8.33%

연도   정부 전체 12월 집중도
2020   10.09%
2024   10.02%
2025   10.11%
```

정부 전체 평균은 균등 대비 +1.7%p로 약함. 그러나 분기 단위로 보면 다른 그림:

```
연도   Q1     Q2     Q3     Q4
2024   27.5   25.1   24.0   23.4
2025   27.0   24.8   26.8   21.4
```

Q1이 균등 25% 대비 +2~5%p 초과. **12월 몰아쓰기보다 Q1 빨리쓰기가 더 두드러짐**.

**관찰**: 분야별 12월 집중도 격차가 8배(33% vs 4%). 일부 분야는 명확한 시점 집중을 보이고 일부는 균등 패턴. 단순 비율 분석으로 1차 패턴은 보임.

## 6. 시행착오 1 — 회계거래 분류의 어려움

### 6.1 문제 인식

분야별 12월 집중도 분석에서 **상위 100% 집중 사업들이 모두 회계 거래(전출·예탁·정산)** 임을 발견. 진짜 정책 사업의 게임화를 측정하려면 회계 거래를 분리해야 함.

### 6.2 시도 1 — 키워드 필터 (넓음)

`ACTV_NM`에 "전출/예탁/이전지출/상환/보전/정산" 등 키워드 매칭:
- 회계거래 226개 / 1,492조 (74.5%)
- 정책사업 2,529개 / 512조 (25.5%)

**문제점**: 지방교부세·기초연금 등 정책 본질의 사업이 회계 키워드로 같이 빠짐.

### 6.3 시도 2 — 좁힌 키워드

순수 자금이동만 분리 ("전출금/타계정/예탁/여유자금운용"):
- 회계거래 56개 / 1,053조 (52.5%)
- 정책사업 2,697개 / 951조 (47.5%)

지방교부세 복귀.

### 6.4 시도 3 — 편성목(CITM) 객관 분류

`expenditure_item`의 `CITM_NM` 기반 객관 분류:
```
CITM = '전출금등'              → 회계거래
CITM = '자치단체이전'          → 지방이전(정책)
나머지 (인건비/운영비/출연금/...) → 정책사업
```

결과:
- 정책사업 1,891개 / 243조
- 지방이전(정책) 188개 / 215조
- 회계거래 97개 / 92조

**키워드 vs CITM 일치도 검증**:
- 키워드는 정책으로 분류했으나 CITM은 회계거래로 본 사례 89건 (회계간거래·예비비 등)
- → CITM이 89건의 회계거래를 추가로 잡아냄, 더 정확

### 6.5 핵심 정정 — Q1 빨리쓰기 가설의 약화

CITM 기반 정책사업만으로 분기 집중도 재계산:

```
연도   Q1     Q2     Q3     Q4
2020   25.9   27.9   21.6   24.5
2024   26.3   24.7   24.8   24.2   ← 거의 균등!
2025   26.2   24.4   27.4   22.1
```

**충격적 정정**: 이전 분석의 "Q1 빨리쓰기 30~35%"는 **자치단체이전(지방교부세) 분기 분배 패턴**의 영향이었음. 정책사업만 보면 Q1 26%로 균등에 가까움.

### 6.6 시행착오의 학술적 의의

분류 노이즈에 따라 결과가 크게 흔들림은 단순 비율 분석의 한계를 명확히 보여줌. **더 강건한 분석 방법론으로 주기성 분해(FFT + STL + Coherence)** 도입 필요.

## 7. 가설 확장 — 주기성 분석

### 7.1 동기

단순 12월 집중도는 한 점만 측정. 시계열 전체의 주파수 성분을 분해하면:
- 정상 정책 흐름 = 트렌드 위주, 약한 계절성
- 게임화된 사업 = 강한 12월/Q1 등 특정 주기 amplitude

이는 분류 노이즈에 강건하고 학술적으로 단단한 측정 방식.

### 7.2 분석 도구

| 도구 | 출처 | 적용 |
|---|---|---|
| FFT | Cooley-Tukey (1965) | 주파수 amplitude 측정 |
| STL Decomposition | Cleveland et al. (1990) | Trend + Seasonal + Residual 분해 |
| Coherence | Welch's method | 두 시계열의 주파수 영역 상관성 |
| Hierarchical Clustering | scipy | Coherence 매트릭스 → 자동 그룹화 |
| Mann-Whitney U Test | 비모수 검정 | 분포 차이 통계 검정 |

### 7.3 3-Tier 분석 구조

| Tier | 단위 | 도구 | 목적 |
|---|---|---|---|
| **Tier 1** | 분야 16개 | STL + FFT | 거시 비교, 시각화 |
| **Tier 2** | 부문 76개 | Coherence + Clustering | 주기적 연관성 발견 |
| **Tier 3** | 단위사업 ~1,600개 (100억+) | 게임화 지수 분포 | 정량 검정 |

## 8. 핵심 발견 — 분야별 게임화 격차 6배

### 8.1 Tier 1: 분야별 연주기 강도

`amp_12m_norm` = 연주기(12개월) FFT amplitude / 평균 — 정규화된 진폭

```
High-Goodhart (강한 주기)         Low-Goodhart (정상)
─────────────────                ─────────────────
과학기술        0.746              국방          0.172
국토·지역      0.607              사회복지      0.166
환경           0.601              공공질서      0.137
보건           0.537              일반·지방행정  0.124
문화·관광      0.467
교통·물류      0.449
산업·중기      0.416
```

**관찰**: 분야별 격차 6배 (0.124 vs 0.746). 인프라·환경·R&D 분야가 강한 주기, 의무 지출 분야(국방·복지·일반행정)가 정상.

### 8.2 구조적 해석

| | High-Goodhart 분야 | Low-Goodhart 분야 |
|---|---|---|
| 사업 형태 | 사업 단위 분할 (건설·R&D·보조금) | 자동 정산·이전 (인건비·교부금·연금) |
| KPI 적용 | 사업별 평가 강함 | 의무 지출 (평가 약함) |
| 행정 재량 | 큼 (시점·금액 조정) | 작음 |
| 대표 사업 | 건설 발주, R&D 출연, 보조금 | 공무원 인건비, 기초연금, 지방교부세 |

→ **굿하트에 취약한 분야 = 사업 단위 분할 + KPI 평가 + 행정 재량 큼** (3박자 가설)

### 8.3 Tier 2: "주기적 연관성"의 자동 발견

76개 부문에 대해 12월 주기 coherence 행렬을 산출하고 hierarchical clustering 적용 → 8개 클러스터로 분해.

가장 흥미로운 클러스터:
```
C6 클러스터 (3개 부문) — 같은 주기 패턴:
  - 공적연금
  - 보건의료
  - 중소기업및소상공인육성
```

## 9. 메인 가설 정식화

```
[H1] 행정 메커니즘
  Game_i = f(사업단위분할도_i, KPI평가강도_i, 행정재량_i)
  
[H2] 정책 효과 (Decoupling)
  ∂Outcome_i / ∂Game_i < 0
  (게임화 강도가 높을수록 결과지표 개선이 더디다)
```

### 9.1 H1 메커니즘 회귀 — 출연 의존도가 게임화를 예측 ★

**모델**:
```
gaming_it = α + β1·chooyeon_pct_it + β2·operating_pct_it + β3·direct_pct_it + ε
```

**Model 1 (단순 회귀, N=90, 분야×연도 패널)**:
| 변수 | 계수 (β) | SE | p값 | 유의 |
|---|---:|---:|---:|---|
| 상수 | +0.388 | 0.084 | <0.001 | *** |
| **출연금 비중** | **+0.375** | **0.132** | **0.005** | **★★★ 1% 유의** |
| **운영비 비중** | **−1.231** | **0.506** | **0.015** | **★★ 5% 유의** |
| 직접투자 비중 | +0.396 | 0.458 | 0.387 | - |
| R² | 0.271 |  |  |  |

**Model 2 (+ 분야 FE + 연도 FE, R²=0.807)**:
- 분야 더미가 게임화 차이의 대부분(80%)을 흡수
- 출연금 효과는 분야 내 시계열에서 사라짐 → 메커니즘은 **분야 간 단면(cross-section)**에서 작동

**해석**:
- **분야 간**: 출연 의존도 1%p ↑ → 게임화 강도 +0.00375
- 사회복지(출연 1%) vs 과학기술(출연 92%)의 게임화 차이 0.34는 출연 비중 차이 91%p × 0.00375 ≈ 0.34 — **거의 정확히 일치**
- 즉 H1 메커니즘 가설("게임화 ∝ 출연 의존도") 정량 입증

**구조적 의미**:
- 게임화는 **분야의 사업 형태**(출연 위탁 vs 직접 집행)에 의해 결정됨
- 분야 내 단기 시계열 변동은 작고, 격차는 사업 구조에서 비롯
- 정책 함의: "분야별 게임화 차이는 행정 재량의 산물 + 위탁 구조의 반영"

### 9.2 H1의 종합 입증

- 분야별 amp_12m_norm 6배 격차 ✓ (Tier 1 결과)
- 사업 형태와의 상관관계 (인프라·R&D > 의무지출) ✓
- 사업 규모와의 비선형 관계 (작은 사업 + 거대 사업 모두 게임화) ✓
- **출연 의존도와의 회귀계수 β=+0.375 (p=0.005)** ✓ ★

**사회복지만 강한 음의 상관**. 다른 분야는 양 또는 표본 부족.

### 10.4 분야별 이질성의 해석

단순 H2(전 분야 음의 상관) 가설은 **반박**. 그러나 분야별 이질성 자체가 의미 있는 발견:

**해석 1 — 사회복지의 음의 상관(ρ=−0.77)**:
- 게임화 강도 ↑ → 자산 지니계수 ↓ (불평등 완화)
- 사회복지의 게임화는 **정기 지급/자동 정산 패턴**의 발현 (의료급여·연금 등). 정기성이 강할수록 저소득층 보전 효과가 안정적으로 작동
- 즉 "게임화"가 항상 비효율을 의미하지 않음 — **자동화된 분배 구조**의 표현일 수도

**해석 2 — 보건·산업·과기의 양의 상관**:
- 게임화 강도 ↑ → 결과 ↑
- 게임화 적응이 잘 된 분야는 행정 운영도 안정적 → 결과 지표도 함께 좋아짐
- "게임화 = 비효율" 단순 도식 부정. **게임화에는 두 종류**: (a) 굿하트형 누수, (b) 정기성/안정성의 발현

**해석 3 — 분야 단면 상에서**:
- 게임화 격차 6배는 H1로 입증. 그러나 게임화의 의미는 분야마다 다름
- 사회복지(0.166, 낮음) vs 인프라·R&D(0.4~0.75, 높음)의 차이는 **사업 형태(이전 vs 분할)**의 반영

### 10.5 한계

- 표본 매우 작음 (분야 5개 × 평균 4년 차분 = 18셀)
- 코로나 충격(2020~2022) 미통제
- 환경·교통·국토 outcome 부재
- 우리 게임화 지수가 **굿하트 효과 vs 정상 운영 패턴**을 구분하지 못함
- 통계적 유의 미달, 이질성 발견은 **정성적 해석** 수준

### 10.6 보고서·PPT 메시지

```
H1 (메인): 분야별 게임화 격차 6배 — 사업 형태와 KPI 압력의 함수
H2 (보조): 게임화-결과 관계는 분야별 이질성을 보임
          - 사회복지: 게임화 = 자동 분배 → 결과 개선 (음 상관)
          - 보건·산업: 게임화 = 운영 안정 → 결과 양호 (양 상관)
          → "게임화의 의미는 분야 형태에 따라 다르다"
```

이 메시지는 "굿하트 = 비효율" 단순 도식보다 학술적으로 정교하며, case 우수작들의 단순 가설 검증을 넘어 **이질성 발견** 수준까지 도달함.

### 11.7 분야 시그니처 비교 — 양극·유사 쌍

분야 12피처 평균 z-score → cosine + amp_12m KS-test:

- **분야 쌍 KS-test (105 pair) 중 29 pair에서 p<0.001** → 분야 간 amp_12m 분포 차이가 통계적으로 매우 유의
- **양극단 (시그니처 정반대)**:
  - 산업·중소기업·에너지 ↔ 공공질서및안전 (cos = −0.92)
  - 국방 ↔ 산업·중소기업·에너지 (cos = −0.69)
  - KS 최대: 국방 vs 국토및지역개발 = **0.59** (분포 거의 별개)
- **유사 쌍 (시그니처 거의 동일)**:
  - 교통및물류 ↔ 국토및지역개발 (cos = **0.91**) — 둘 다 인프라 공사형
  - 일반·지방행정 ↔ 통일·외교 (cos = 0.85)
  - 공공질서및안전 ↔ 통일·외교 (cos = 0.81) — 둘 다 운영비·Q4 집중

### 11.8 H3의 의의

1. **분야 단위 회귀(N=15)와 활동 단위 임베딩(N=1,641)이 같은 결론**을 가리킴 → H1 메커니즘(출연금→게임화) 강건성 ↑
2. "정상 사업"도 15개 미시 패턴으로 분해됨 → 분야 평균을 가르는 진짜 단위는 **사업 형태**, 분야 라벨이 아님
3. 부처는 행정형·사업형·분기말집중·직접투자·출연금 5개 체질로 묶임 → 부처별 게임화 차이는 **개별 부처 특질**이 아니라 **구조적 그룹 효과**
4. 분야 쌍 KS-test의 28%가 강력 유의 → 분야 간 게임화 패턴 차이는 무작위 아님

## 12. H4 — Mapper(TDA) 위상 + 활동 단위 outcome 디커플링

### 12.1 동기

H3는 평면 클러스터까지. 문제 두 가지가 남는다.
- **위상 검증**: H3의 3+15 클러스터가 진짜로 분리된 구조인지, 아니면 연속체에서 임의로 자른 것인지?
- **outcome 연결**: H2(분야 5개)의 디커플링이 활동 단위에서도 보이는지?

Mapper(Topological Data Analysis)는 두 질문을 동시에 다룬다. UMAP 2D를 filter로 쓰고 Cover×Cluster로 노드를 만든 뒤 연결성을 보면, 클러스터가 **위상적으로 분리된 컴포넌트**인지 **하나의 큰 매니폴드 위의 부분**인지 알 수 있다.

### 12.2 Mapper 구성

```
filter   : UMAP 2D (H3와 동일 좌표 — 재현성 ↑)
cover    : 18×18 큐브, 50% overlap
cluster  : 각 큐브 내부 DBSCAN(eps=0.45, min_samples=4)
출력     : 26 nodes / 34 edges
```

### 12.3 위상 통계

| 지표 | 값 | 의미 |
|---|---:|---|
| 0차 베티수 (component) | **8** | 8개의 분리된 사업 원형 군 |
| 1차 베티수 (loop) | **7** | 메인 컴포넌트 안 닫힌 경로 7개 |
| 분기점 (deg≥3) | 7 | 사업 원형이 만나는 결절점 |
| 최대 컴포넌트 | 8 nodes / 14 edges | 정상 사업 원형들이 연속적으로 연결 |
| 도수 분포 모드 | deg=3 (15 nodes) | 트리 + 약한 loop 구조 |

→ **H3 클러스터 1(출연금형)·인건비형 일부는 별도 컴포넌트로 떨어져나옴**: 위상적으로도 분리. 정상 사업(C2)은 한 큰 컴포넌트 안에서 7개 loop를 만들며 미시 패턴들이 연속적으로 변형됨 — H3의 sub-cluster 15개가 toroidal/branched 구조 위에 있다는 신호.

### 12.4 활동 단위 outcome 디커플링 (H2 확장)

분야 단위 KOSIS outcome 5개를 클러스터별 활동 분포로 가중평균 → 클러스터 시계열 → 1차 차분 상관:

| outcome | C0 인건비 | C1 출연금 | C2 정상 |
|---|---:|---:|---:|
| industry_production_index | +0.69 | **+0.74** | +0.70 |
| life_expectancy | +0.13 | **−0.88** | −0.19 |
| wealth_gini | **−0.82** | +0.26 | **−0.77** |

핵심 신호:
1. **wealth_gini (사회복지) — C0/C2는 −0.8 음 상관 (자동분배 효과), C1은 +0.26 (반대 부호)**: 출연금형 사업의 게임화는 빈곤 격차를 *완화하지 않는다* (다른 사업 형태와 정반대).
2. **life_expectancy (보건) — C1만 −0.88 강한 디커플링**: 출연금형 사업이 활성화될수록 보건 outcome은 *오히려 악화* 추세. 출연금이 보건 분야 활동의 1%에 불과한데도 강한 음 신호 → 표본 작아 신중하게 해석하지만 H1 메커니즘과 일관 (출연금형이 굿하트 누수가 클 위험).
3. **industry — 세 클러스터 모두 +0.7대**: 산업 outcome은 게임화와 같이 움직이는 수반 효과. 디커플링 약함.

### 12.5 시계열 패턴

```
amp_12m 가중평균 시계열 (분야 가중):
  cluster 0: 0.27 → 0.36 (선형 상승)
  cluster 1: 0.29 → 0.45 (2022 spike, 출연금형의 외생 충격)
  cluster 2: 0.25 → 0.36 (선형 상승)
```

→ C1만 2022년 spike — 출연금형 사업이 외부 충격(예: R&D·교육 출연 확대, 코로나 회복)에 더 반응적. 다른 클러스터는 점진적 상승.

### 12.6 H4의 의의

1. **H3 클러스터의 위상적 검증**: 출연금형(C1)·인건비형(C0)은 정상 사업(C2)과 *분리된 컴포넌트*에 속함 — 임의 분할이 아니라 위상적 단절.
2. **H2(분야 5개)의 활동 단위 확장**: 클러스터별 outcome 차분 상관이 H3 출연금형에서만 부호 반전 (wealth_gini +0.26 vs 다른 −0.8) → "출연금 의존 사업의 게임화가 진짜 굿하트일 가능성"이라는 가설을 *분야 라벨 없이* 도출.
3. **표본의 한계 명시**: outcome 시계열 4~6년, 분야 5개라 통계적 검정력은 낮음. 이는 **신호의 방향**이지 효과 크기 추정 아님.

## 13. 한계 및 주의

### 11.1 데이터 한계
- **편성 vs 결산 차이**: 본예산·추경·월별 집행 보유, 결산(이월·불용)은 일부만
- **5년 시계열**: 60개월이지만 1년 주기는 5회만 관측 → 통계적 검정력 한계
- **분야 분류 의존성**: 16대 분야는 행정 분류이지 행동 분류가 아님
- **Outcome 매핑 부분성**: 16개 분야 중 5개에만 outcome 매핑. 환경·교통·국토는 KOSIS 주제별통계에 적합한 거시 지표 부재

### 11.2 인과추론 한계
- 한국 단일 정부 정책 → DID 통제군 부재
- 처치(KPI 도입) 시점 명확하지 않음 (2007년 디지털예산회계 도입 후 점진적)
- 역인과·공변량 누락(코로나 등) 통제 어려움
- H2 검증의 표본 부족 (분야 5 × 평균 4년 차분 = 18셀)

### 11.3 측정 한계
- "KPI 평가 강도"·"행정 재량"의 직접 측정 불가 → 분야 더미 사용
- amp_12m_norm은 12개월 주기 진폭만 측정 (다른 주기는 별도)
- 게임화 지수가 **굿하트 누수**와 **정상 운영 안정성**을 구분 못 함

### 11.4 정책 함의의 신중함
- 게임화가 강하다 ≠ 행정이 비효율적이다 (사회복지 사례에서 반증)
- 일부 분야의 시점 집중은 사업 특성(공사 완료 시점·계절성·법정 의무)에서 비롯
- 정책 제언은 게임화 식별 + 결과변수 Decoupling 입증 후에만
- 사회복지 음의 상관은 표본 5점이라 신중한 해석 필요
